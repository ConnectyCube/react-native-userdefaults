# react-native-userdefaults

An wrapper voor iOS UserDefaults

## Installation

```sh
npm install react-native-userdefaults
```

## Usage

```js
import Userdefaults from "react-native-userdefaults";

// ...

const result = await Userdefaults.multiply(3, 7);
```
